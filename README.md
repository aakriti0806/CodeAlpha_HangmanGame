# 🎮 Hangman Game — CodeAlpha Internship Task 1

A simple text-based Hangman game built with Python.

## 📌 Features
- Random word selection from predefined list
- 6 attempts before game over
- Live display of guessed letters and remaining lives
- Win/loss detection

## 🛠 Technologies Used
- Python 3
- random, datetime modules

## ▶️ How to Run
python hangman.py

## 📚 Concepts Used
- while loop, if-else, lists, strings, random module
